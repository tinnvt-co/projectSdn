# Dormitory Management System - Quản Lý Ký Túc Xá

Hệ thống quản lý ký túc xá sử dụng MERN Stack.

## Tech Stack

- **Frontend:** React.js (Vite)
- **Backend:** Express.js
- **Database:** MongoDB (Mongoose)

## Cài đặt & Chạy

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Cấu hình

Tạo file `backend/.env` với nội dung:
```
PORT=5000
MONGO_URI=mongodb://localhost:27017/dormitory_management
JWT_SECRET=your_jwt_secret_key_here
```
